# Markdown Cheat Sheet

## Text Formatting
- **Bold**: `**text**` or `__text__`
- *Italic*: `*text*` or `_text_`
- `Code`: `` `code` ``
- ~~Strikethrough~~: `~~text~~`

## Lists
- Unordered: `- Item` or `* Item`
- Ordered: `1. Item`

## Links and Images
- Link: `[Google](https://www.google.com)`
- Image: `![Alt text](image.jpg)`

## Code Blocks
- Inline: `` `code` ``
